The Key Promoter X has now reached over 320.000 downloads. It is top-rated on the JetBrains site, and since it is entirely open-source, other plugin developers can browse through the code to understand how things are implemented.

Latest changes

  • Your shortcut statistic is now kept local to prevent merging conflicts when using IDEA's "Settings repository" to keep configuration in sync on multiple workstations (Issue 40)
  • Add features that let's you turn off KPX in Presentation and Distraction Free Mode (Issue 39)
  • Fix instances where the KPX would not recognize a mouse-click. This replaces a hack that worked around an issue in IDEA with still another hack that hopefully works better.
