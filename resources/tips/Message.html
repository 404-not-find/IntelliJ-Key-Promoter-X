Since its first release, the Key Promoter X has now more than 240.000 downloads. It is top-rated on the JetBrains site, and since it is entirely open-source, other plugin developers can browse through the code to understand how things are implemented.

The plugin is not for me or a by-product of something I would do anyway. It is for you who want to learn shortcuts in IntelliJ IDEA, and I give away some of my free time to keep it up to date and fix bugs. If you appreciate my efforts, then please donate some money. And don't forget to star the GitHub repository and leave a rating on the JetBrains plugin site.
